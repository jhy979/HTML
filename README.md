# HTML
### html 문법에 대하여 알아봅시다.

## 1. 빈 태그
```html 
<div> </div>
```
- 원래 이게 기본인데
```html
<meta charset="UTF-8">
<link rel="stylesheet" href="./main.css">
```
- 같이 끝 태그가 없는 경우를 빈 태그라고 합니다.
```
<태그> vs <태그/>
```
<태그> : HTML 1/2/3/4/5 버젼 <br/>
<태그/> : XHTML, HTML5 버젼 (더 안전해요~)

## 2. attribute 기능의 확장
> <태그 속성="값"> 내용 </태그>
```html
<img src= "~~~~" alt="nope"/>
```
- 위의 img태그 같은 경우에는 src, alt같은 것들이 attribute
